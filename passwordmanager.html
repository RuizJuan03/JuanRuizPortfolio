Python Password Manager

Project Summary

This project is a password manager tool designed to securely store user credentials. It utilizes a MySQL database for data storage, with Python for backend processing. Security is a top priority in this project, with passwords being hashed and salted to ensure their protection. Passwords are not only hashed and salted to ensure their protection but also monitored through Splunk logging at each critical interaction point, like when an account is created, viewed, updated, and deleted.

Technical Details

The Python Module "HashLib" uses the SHA-256 hashing algorithm combined with a unique salt thats 16 bytes for each password, ensuring robust security. I implemented advanced Python techniques for secure database connections and efficient data handling by storing database credentials in a json file in a different file.

Project Challenges

While developing this project one of the biggest challenges I faced was using Splunk's proprietary Splunk Processing language (SPL) designed for searching, filtering, and manipulating data indexed by Splunk. To get familiar with SPLs syntax I decided to watch YouTube videos on it which not only helped me learn more about it but also got me more familiar with Splunks UI.
